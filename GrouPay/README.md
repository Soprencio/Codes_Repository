# GrouPay
GrouPay es una plataforma en Java para la gestión financiera de familias y PyMEs. Permite administrar cuentas personales y fondos comunes, facilitando el pago de facturas y transferencias. Destaca por su módulo de deudas, que rastrea saldos y vencimientos, ofreciendo cálculos de presupuesto y alertas de fondos para un control económico total.
