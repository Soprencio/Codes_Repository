# Repositorio Manuel Ponce

En este repositorio se pueden encontrar múltiples códigos Java, C y Python. Los proyectos más grandes e importantes son CINEMARX (Java), BancoADN (Java), GrouPay (Java), SUBE-Definitive (QT-C/C++), Problemas Atencion (QT-C/C++) y Sokoban (Python)
Los 3 proyectos Java fueron realizados en equipo en forma de proyectos de distinta modalidad y requerimientos.
